[JPEGSnoop](http://sourceforge.net/projects/jpegsnoop/)
a free Windows-only application that can detect whether an image has been edited. Despite its name it can open AVI, DNG, PDF, THM and embedded JPEG files. It also retrieves metadata including: date, camera type, lens settings, etc.For Mac users, JPEGsnoop works within CrossOver Mac.
